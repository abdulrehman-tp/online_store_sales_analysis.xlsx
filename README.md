# online_store_sales_analysis.xlsx
# Excel Sales Dashboard Project

## 📊 Project Overview

This project analyzes online store sales data using Microsoft Excel.
It includes data cleaning, formula calculations, pivot tables, and dashboard creation.

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Charts (Bar Chart)

---

## 📁 Dataset

The dataset contains:

* Order details (Order ID, Date, Customer, City)
* Product information (Category, Product)
* Sales metrics (Sales, Quantity, Discount, Profit)

---

## 📌 Key Features

* Profit calculated using Excel formulas
* Created Pivot Tables for analysis
* Built interactive charts:

  * City vs Profit
  * Category vs Sales
* Designed a clean dashboard with KPIs:

  * Total Sales
  * Total Profit
  * Average Discount

---

## 📈 Key Insights

* Electronics category generates highest sales and profit
* Bangalore city has highest profit
* Higher discount tends to reduce profit

---

* Data Cleaning
* Data Analysis
* Pivot Tables
* Data Visualization
* Dashboard Creation

This project demonstrates basic to intermediate Excel skills required for a Data Analyst role.
